🫐 Blackcap Pi

```

A Raspberry Pi + e-ink kitchen display for weekly menus, recipe mode, and smart icon footers.

Blackcap Pi turns a simple published Google Sheet into a calm, always-on menu display with automatically generated visual icons based on your weekly menu.

The name comes from the Blackcap (Rubus occidentalis), a type of black raspberry that tastes like a traditional raspberry but has a deep, dark color—fitting for a project built around the black-and-white simplicity and efficiency of an e-ink display. Also, “Inky Pi” was already taken.

Curious about the fruit? Learn more about Rubus occidentalis .

View on GitHub ```

Automatically generated icons based on weekly menu content.

📱 Live Interaction

Control the display in real time from your phone or tablet.

📅 Automated Menu Mode from a published Google Sheet
